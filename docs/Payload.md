# Payload
Payload is an object which is used as a needle to find something in other object.
<br>

### We have the following Payload implementations:
<ul>
    <li> StandardBufferPayload: This implementation is used to match Buffer payload against another Buffer. </li>
    <li> StandardRegexPayload: This implementation is used to match Regex String against another String. </li>
    <li> StandardHexPayload: This implementation is used to match Regular Expressions (Regex) against another Payload using </li>
</ul>
